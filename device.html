设备（device）

设备（device）

电话

addPhoneContact

  • 新增手机通讯录联系人

  • 无传入参数

传入参数 必填 说明
name 是 姓名
phoneNumber 是 手机号码
  • 回调参数：
回调参数 说明
result 0成功
  • 说明：该方法只打开联系人新增页面，并替用户写好联系人姓名和电话号码，具体用户填什么，是否新增成功，APP无法得知，只能知道用户已经操作完成回到当前页面。

selectPhoneContact

  • 从手机通讯录选择一个联系人

  • 无传入参数

  • 回调参数：

回调参数 说明
name 联系人姓名
phoneNumber 联系人电话号码

makePhoneCall

  • 调用系统打电话

  • 传入参数：

传入参数 类型 必填 说明
phoneNumber string 是 需要拨打的手机号码
  • 回调参数：无

剪贴板

setClipboardData

  • 设置系统剪贴板的内容

  • 传入参数：

传入参数 类型 必填 说明
data string 是 传入剪切板的内容

  • 回调参数：无

  • 说明：调用成功后会toast提示“内容已复制”

getClipboardData

  • 获取系统剪贴板的内容

  • 传入参数：无

  • 回调参数：

回调参数 类型 说明
data string 获取的剪切板的内容

网络状态

onNetworkStatusChange

  • 监听网络状态变化事件

  • 传入参数：无

  • 回调参数：

值 说明
wifi wifi网络
cellular 蜂窝网络
none 无网络
  • 说明：使用startNetworkStatusChange开始监听，开始监听后在原生调用此方法进行实时回调，使用结束需要调用stopNetworkStatusChange停止监听

startNetworkStatusChange

  • 取消监听网络状态变化事件

  • 传入参数：无

  • 回调参数：无

stopNetworkStatusChange

  • 取消监听网络状态变化事件

  • 传入参数：无

  • 回调参数：无

getNetworkStatus

  • 获取网络类型

  • 传入参数：无

  • 回调参数：

回调参数 类型 说明
networkType string 网络类型，值为2g,3g,4g,wifi,unknown,none
  • 说明：networkType的合法值
值 说明
wifi wifi网络
cellular 蜂窝网络
none 无网络，未识别也传none
