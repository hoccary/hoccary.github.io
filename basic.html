基础（basic）

基础（basic）

系统

isApiExist

  • 判断API是否可调用

  • 传入参数：

传入参数 说明
apiName api的名字
group group名字
  • 回调参数：
回调参数 说明
result "0"可以使用，"1"无法找到该api，"2"无权限

getSystemInfo

  • 获取系统信息

  • 无传入参数

  • 回调参数：

回调参数 说明
platform 设备平台，iOS或Android
brand 设备品牌及型号，iPhone X
version APP版本号，1.0.0
system iOS或Android系统版本，13.0.1
benchmarkLevel 设备性能等级，0、1、2、3，数值越高性能越好，性能差时可以关闭动态效果和其他费时操作
  • 说明： 以上字段，识别失败返回 nat-unknown

生命周期

onDocumentLoaded

  • 每次webview加载完成时会调用

  • 传入参数

传入参数 说明
data 前一个页面传来的参数。通过前一个页面open方法data传递来的参数
  • 无回调参数

onPageShow

  • webview切回前台事件之前调用

  • 无传入参数

  • 无回调参数

onPageHide

  • webview切到后台事件之前调用

  • 无传入参数

  • 无回调参数

navigateBack

  • 关闭当前webview返回上一页

  • 无传入参数

  • 无回调参数

relaunch

  • 重载当前页面

  • 传入参数

传入参数 说明
url 重载时的入口url，传空则会使用默认的入口url
  • 无回调参数
