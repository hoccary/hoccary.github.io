数据缓存（cache）

缓存

每个页面都有自己的一个缓存目录，用于存放持久化的文件及匹配，比localStorage存储时间更长，且可以不依赖webview的缓存机制，不会被自动删除，除非用户主动清除缓存。

加密

encyptData

  • SM4算法加密字符串，如果是图片等其他文件，需要base64编码为字符串，然后加密

  • 传入参数：

传入参数 说明
data 要加密的字符串
isLocal 是否是本地加密，0：本地加密，使用APP自带的秘钥，如果传0或空，下列字段都不用传；1：需和SAP秘钥保持一致，需提前在SAP配置
clientid 需传clientid
trantm 公共报文头的trantm字段
keyversion 报文头里的keyversion字段
  • 回调参数：
回调参数 说明
data 加密后的字符串

decyptData

  • 将数据存储在本地缓存中指定的key中

  • 传入参数：

传入参数 说明
data 要解密的字符串
isLocal 是否是本地解密，0：本地解密，使用APP自带的秘钥，如果传0或空，下列字段都不用传；1：需和SAP秘钥保持一致，需提前在SAP配置
clientid 需传clientid
trantm 公共报文头的trantm字段
keyversion 报文头里的keyversion字段
  • 回调参数：
回调参数 说明
data 解密后的字符串

数据

setStorage

  • 将数据存储在本地缓存中指定的key中

  • 传入参数：

传入参数 说明
key 本地缓存中指定的key
value 字符串
  • 无参数回调

removeStorage

  • 从本地缓存中移除指定key
  • 传入参数：
传入参数 说明
key 本地缓存中指定的key
  • 无参数回调

getStorage

  • 从本地缓存中异步获取指定key的内容
  • 传入参数：
传入参数 说明
key 本地缓存中指定的key
  • 回调参数：
传入参数 说明
value 返回字符串，无值或无key返回空字符串

clearStorage

  • 清理本地数据缓存
  • 无传入参数：
  • 回调参数：
传入参数 说明
result 0成功 1失败

文件

saveFile

  • 保存文件到本地
  • 传入参数：
传入参数 说明
obj 文件内容
filePath 文件路径 + 文件名, String，例：'/kjhk/header.jpg'
  • 回调参数：
传入参数 说明
result 0 存储成功 1存储失败

readfile

  • 读文件

  • 传入参数：

传入参数 说明
filePath 文件路径
  • 回调参数：
传入参数 说明
data 文件内容
result 执行结果，0成功1失败

removeFile

  • 删除本地缓存文件
  • 传入参数：
传入参数 说明
filePath 文件路径
  • 回调参数：
传入参数 说明
result 0 删除成功,无文件也返回0； 1删除失败

isFileExist

  • 判断文件/目录是否存在
  • 传入参数：
传入参数 说明
filePath 文件的路径（本地）
  • 回调参数：
传入参数 说明
result 0 存在 1 不存在
