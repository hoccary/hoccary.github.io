多媒体（media）

多媒体

图片

saveImageToPhotosAlbum

  • 保存图片到系统相册
  • 传入参数：
传入参数 说明
image Data二进制数据
  • 回调参数：
回调参数 说明
result 0 保存成功 1保存失败

previewImage

  • 在新页面中全屏预览图片
  • 传入参数：
传入参数 说明
image Data二进制数据
  • 无参数回调

compressImage

  • 压缩图片接口，可选压缩质量
  • 传入参数：
传入参数 说明
image Data二进制数据
quality 质量，0 - 1，1为无损
  • 回调参数：
回调参数 说明
data Data二进制数据

chooseImage

  • 从本地相册选择图片或使用相机拍照获取图片

  • 无传入参数

  • 回调参数：

回调参数 说明
data Data二进制数据

视频

chooseVideo

  • 拍摄视频或从手机相册中选视频页面

  • 无传入参数

  • 回调参数：

回调参数 说明
data 视频Data二进制数据
  • 说明： 存储视频文件使用cache里的saveFile方法

录音

startRecord

  1. 开始录音
  2. 无传入参数
  3. 回调参数：
回调参数 说明
result 0 正在录音 1 设置类型失败 2 初始化动作失败 3录音失败

stopRecord

  1. 停止录音
  2. 无传入参数
  3. 无参数回调

回调参数 说明

pauseVoice

  • 暂停播放语音文件
  • 传入参数
传入参数 说明
code 0 暂停 1 继续播放
  • 回调参数：
回调参数 说明
result 0 成功 1失败

stopVoice

  • 停止播放语音文件
  • 无传入参数
  • 回调参数：
回调参数 说明
result 0 成功 1失败
